# LaMath

A math interpreter written in PHP

## Quick Try
```console
$ php bin/lamath
>> 5 + 5;
10
```

## Syntax

- The numbers are reperesentated as PHP floats
- A math expression is a step, and the whole line of expressions is a procedure
- The steps must end with a ';'

The available unary operators are:
[]

The available binary operators are:
[+, -, *, **]
+: Addition
-: Subtraction
*: Multiplication
**: Exponentation
/: Division
